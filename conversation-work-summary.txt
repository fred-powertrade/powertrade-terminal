8. 5-PHASE UX OVERHAUL (2026-03-08)
   Phase 1 — Interactivity & Discoverability:
   - Treemap tiles: Plotly click handler + keyboard (Enter/Space) support
   - Flow feed items: clickable → selectAsset()
   - Funding bar rows: clickable → selectAsset()
   - Header BTC/ETH stats: onclick + onkeydown
   - Bubble chart: canvas click hit-detection with radius test

   Phase 2 — Side Detail Panel:
   - openDetailPanel(sym): token name/price/change, KPI grid, quick-action buttons
   - closeDetailPanel(): removes .open class, smooth scroll

   Phase 3 — Enriched Deep-Dive Modal:
   - LightweightCharts candlestick price chart (30-day synthetic OHLCV)
   - Alpha Intelligence Summary: plain-English narrative from Funding/RSI/IVRank/GEX
   - Actionable Playbook: LONG/SHORT bias, entry trigger, 1σ stop, 2σ target
   - Position sizing guidance via calcExpectedMove()
   - Quick Trade Actions footer: Market Buy/Sell, Open Orderbook

   Phase 4 — Explainability & Trust:
   - Explain modal: Sample Size (N) + Precision % columns
   - Confidence banner (green/red) based on precision < 50% or N < 20
   - ESTIMATED/LIVE data-source tags on chart sections
   - ℹ Explain buttons on Fair Value Consensus and IV Rank

   Phase 5 — Performance & Memory:
   - Plotly.purge() before newPlot to prevent memory leaks
   - Focus trap in deep-dive modal (Tab/Shift+Tab wrapping)
   - Focus returned to opener element on modal close
